# C'era una volta a Piossasco

## Obiettivo
Creare un sito web semplice, amatoriale, mobile-first, pensato per i piossaschesi.
Non deve sembrare istituzionale.
Deve essere bello, leggero e facile da usare.

## Cosa include l'MVP
- Home page
- Sezione Storia
- Sezione Curiosita
- Sezione Eventi
- Sezione Pizzerie
- Sezione Planner AI
- Admin panel minimale

## Cosa escludere
- Account utenti
- Community
- Commenti
- Recensioni
- Google Maps API
- Monte San Giorgio
- SEO avanzata
- Analytics avanzate
- Monetizzazione
- Filtri complessi
- Ricerca globale
- Funzionalita enterprise
- Architetture complesse

## Tono e design
- Moderno
- Amichevole
- Locale
- Non tech
- Non istituzionale
- La sezione Storia puo avere un mood piu storico/editoriale

## Target
Piossaschesi e persone dei dintorni

## Lingua
Solo italiano
