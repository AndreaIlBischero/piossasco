# Routes

## /
Home page del progetto.
Deve introdurre il sito e mostrare anteprime delle sezioni principali:
- eventi
- pizzerie
- planner
- storia
- curiosità

## /storia
Pagina dedicata alla storia di Piossasco.
Deve contenere:
- timeline verticale/interattiva mobile-first
- sezione per temi
- mood più editoriale/storico

## /curiosita
Pagina separata con curiosità locali.
Contenuto semplice, leggibile e piacevole da scorrere.

## /eventi
Pagina con lista eventi.
Requisiti:
- ordine cronologico
- card semplici
- immagine
- titolo
- data
- luogo
- descrizione breve
- link fonte
- archivio eventi passati

## /pizzerie
Pagina con elenco delle pizzerie inserite manualmente.
Ogni card mostra:
- nome
- categoria
- foto
- indirizzo testuale
- pulsante/link verso Google Maps

## /planner
Pagina con assistente semplice per generare 2 proposte di giornata.
Input tramite pulsanti rapidi o form semplice.
Output:
- timeline
- elenco tappe
- tono naturale

## /admin
Pagina admin minimale protetta da password.
Funzioni:
- lista eventi
- elimina evento
- lista pizzerie
- aggiungi/modifica pizzeria
- visualizza contenuti storia/curiosità
